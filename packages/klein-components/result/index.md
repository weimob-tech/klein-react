### Result Props

| 参数 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :----- |
| icon      | 设置按钮的图标 | `ReactNode`  | -      |
| extra      | 操作区 | `ReactNode`  | -      |
| subTitle      | subTitle 文字 | `ReactNode`  | -      |
| title      | title 文字 | `ReactNode`  | -      |
| className      | 自定义className | `string`  | -      |
| style      | 自定义style | `CSSProperties`  | -      |
