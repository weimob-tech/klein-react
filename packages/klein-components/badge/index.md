# 徽标 Badge

图标右上角的圆形徽标数字一般出现在通知图标或头像的右上角，用于显示需要处理的消息条数，通过醒目视觉形式吸引用户处理

### Badge Props

| 参数 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :----- |
| dot      | 不展示数字，只显示一个小红点 | `boolean`  | -      |
| color      | 自定义背景颜色 | `string`  | -      |
| count      | 展示的数字 | `ReactNode`  | -      |
| size      | 尺寸大小 | `"small"`  | -      |
| offset      | 设置位置的偏移量 | `number[]`  | -      |
| className      | 自定义className | `string`  | -      |
| border      | 微标描边样式 | `string`  | -      |
